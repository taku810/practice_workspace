# Git修行の館

## ここは？

Gitの修行をするための場所です。
Gitマスター目指して、頑張りましょう！

## 修行内容

* 基本の型：コミット、プッシュ、プル
* 応用の型：ブランチ、マージ、コンフリクト
* 奥義：チーム開発

## 修行方法

1.  まずは入門！
    ```bash
    git clone https://github.com/taku810/practice_workspace.git
    ```
2.  `src/`フォルダで修行開始！
